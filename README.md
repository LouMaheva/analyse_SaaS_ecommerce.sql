# analyse_SaaS_ecommerce.sql
Pipeline Data &amp; Requêtes SQL pour analyser un SaaS E-commerce (Bubble.io). Centralisation des commandes WhatsApp/Facebook à Abidjan, calcul des KPIs (CA, Panier Moyen, Rétention) et Dashboard Tableau d'aide à la décision.

📊 Analyse de Performance Commerciale & Product Analytics (SaaS E-commerce)

Ce projet présente l'architecture de données et les requêtes de Business Intelligence conçues pour analyser l'activité d'une application web (SaaS) développée sur Bubble.io. L'application permet aux e-commerçants à Abidjan de centraliser leurs commandes omnicanales (WhatsApp & Facebook).

🎯 Objectifs du projet :
- Modéliser une base de données relationnelle propre à partir d'exports applicatifs bruts.
- Analyser la performance commerciale globale (Chiffre d'Affaires, Panier Moyen).
- Comparer la rentabilité et les taux d'annulation entre les canaux WhatsApp et Facebook.
- Analyser l'impact logistique des tarifs de livraison par commune (Cocody, Yopougon, Abobo...).
- Mesurer l'engagement et l'adoption du produit par les vendeurs (SaaS Metrics).

🛠️ Technologies utilisées :
- SQL (PostgreSQL / MySQL) : Requêtage complexe, Jointures, Agrégations, Data Quality Audit.
- Tableau Software : Conception d'un dashboard interactif d'aide à la décision.
- Excel / Power Query : Nettoyage (Data Cleaning) et normalisation des encodages de caractères.
